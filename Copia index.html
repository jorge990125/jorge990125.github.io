# Información Adicional

El diseño BlizzCms Estilo blizzlike fue creado pensando en una experiencia visual inmersiva, con elementos que recuerdan la interfaz original de World of Warcraft pero adaptados a las necesidades modernas. Este diseño incluye:

  • Colores y tipografías basadas en el tema clásico de Blizzard.
  • Interfaz optimizada para usuarios nuevos y veteranos.
  • Soporte para personalización de módulos y temas.
  • Compatibilidad con las versiones 1.x de BlizzCMS.

Este estilo está diseñado para proporcionar un balance entre nostalgia y funcionalidad, destacando la esencia del mundo de Azeroth en cada elemento visual.

# Características Servidor

  • Version del Emulador 4.3.4(Cataclysm).
  • Sistema Lua.
  • Sistema Vip.
  • Sistema Lvl boost por token.
  • Sistema de arneas anti empates.
  • Sistema de suscripción(en desarrollo).
  • Sistema de Arenas 1vs1 con mmr.
  • Sistema de crossfaction pvp-pve.
  • Npc Teleport a Capitales, Bandas y Mazmorras.
  • Auto aprendisaje de poderes.
  • Titulos pvp automaticos.
  • Npc de cambio de racial.
  • Chat global comando .world.
  • Transmog custom.
  • Nuevos items para transfiguración.
  • Sistema de vinculación de monturas a la cuenta no al personaje(es decir que las monturas que consigas con un personaje las obtendrán directamente los demás personajes de la cuenta)
  • Zonas iniciales funcionando.
  • Bandas cataclismo funcionando.
  • Mazmorras cataclismo funcionando.
  • Poderes de clases bastante aceptables
  • Npc Lvl boost.
  • Npc rates custom.
  • Npc profesiones mediante token.
